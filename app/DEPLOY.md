# Deploy na Vercel

App: Vite + React + TS (PWA). Consome figurinhas de https://agenciaspace.github.io/cloc-figurinhas

## Vercel
1. Importe o repo `agenciaspace/cloc-figurinhas` na Vercel.
2. **Root Directory:** `app`
3. Framework preset: **Vite** (auto). Build: `npm run build` · Output: `dist`
4. Deploy. Pronto — vira PWA instalável.
