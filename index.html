📲 Como adicionar no WhatsApp

As figurinhas já são 512×512 (.webp), então importam direto. O WhatsApp não deixa adicionar por 1 toque na web — use um app de figurinhas grátis (passo único):

  1. Android: instale Sticker Maker → "Criar pacote" → adicione as figurinhas baixadas → Adicionar ao WhatsApp.
  2. iPhone: instale Sticker Maker Studio → novo pacote → importe os .webp → Add to WhatsApp.
  3. Pra uma figurinha só: baixe (⬇), abra no app de figurinhas e toque em adicionar.
